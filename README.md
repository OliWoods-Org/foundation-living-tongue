> Few-shot speech AI for endangered languages. Elder oral history transcription. Interactive learning.

## The Solution

**SpeechEngine** — Speech-to-text for low-resource languages using few-shot learning\n**OralHistorian** — Transcribe elder oral histories from audio recordings\n**LanguageLearner** — Interactive chatbot trained by community elders\n**DictionaryBuilder** — Documentation tools for grammar and vocabulary\n**TranslationBridge** — Translation between endangered and dominant regional languages
